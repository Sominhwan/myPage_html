자바 언어 기초 프로그래밍 강의  New

Java 강의는 객체 지향 프로그래밍의 기본을 알려주는 강의입니다. 처음 프로그래밍을 접하는 입문자가 듣기에 적합한 강의입니다. 강의료는 무료이며 Java 기초 프로그래밍 강좌는 총 30강으로 구성됩니다.


안드로이드 기초 프로그래밍 강의  New

안드로이드는 리눅스 커널을 기반으로 구글에서 제작하고 있는 스마트폰과 같은 플랫폼의 모바일 운영 체제와 미들웨어 및 중요 애플리케이션이 포함된 소프트웨어 집합입니다. 안드로이드 관련 기초 강의를 듣고 싶다면 클릭하세요.
